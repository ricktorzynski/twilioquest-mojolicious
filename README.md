Install Mojolicious
```
$ sudo -s 'curl -L cpanmin.us | perl - Mojolicious'
```

Generate an app
```
$ mojo generate lite_app myapp.pl
```

Modify script

Start in daemon mode
```
$ ./myapp.pl daemon -l http://*:8767
```
